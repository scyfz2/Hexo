C盘满了如何清理？

Fan2025-12-232025-12-24

Windows C盘清理终极指南

一、First of all, 在刚拿到电脑的时候就多分一点给C盘

要对电脑的C盘进行分区，可以按照以下步骤操作：

  1. 打开磁盘管理：右键点击“此电脑”，选择“管理”，然后点击左侧的“磁盘管理”。
  2. 压缩C盘：在磁盘管理界面中，右键点击C盘，选择“压缩卷”，输入要压缩的空间大小，然后点击“压缩”。
  3. 创建新分区：压缩后会生成一个“未分配”的新空间，右键点击该空间，选择“新建简单卷”，按照向导完成新分区的创建。
  4. 分区建议：建议C盘分配100GB-150GB用于系统（再预留100GB给以后杂七杂八的东西），D盘用于存储个人数据（也可以再加个学习工作盘），E盘用于娱乐内容，F盘用于备份啥的。

✅ 通过以上步骤，你可以成功对C盘进行分区。

二、使用磁盘清理工具

Windows 磁盘清理是最简单的C盘清理工具，可以安全删除临时文件、系统日志、Windows更新文件等。

操作步骤：

  1. Win + R 输入 cleanmgr 回车
  2. 选择 C盘，点击 确定
  3. 点击 清理系统文件，再次选择 C盘
  4. 勾选以下选项：
    • 以前的 Windows 安装（Windows.old）
    • 临时文件
    • 回收站
    • Windows 更新清理
    • 其实都可以删除，我感觉都没有用，你要是害怕就自己挑一下
  5. 点击 确定，开始清理

✅ 效果：释放 5GB-10GB 空间。

三、删除 Windows.old 旧系统文件

如果从Win7/Win8升级到Win10，系统会自动生成Windows.old备份，占用5GB-20GB空间。

操作步骤：

  1. 打开 磁盘清理（参考方法二）
  2. 选择 清理系统文件
  3. 勾选 以前的Windows安装，点击 删除

✅ 效果：释放 5GB-10GB 空间。

四、清理 Windows 更新缓存

Windows 更新后，会残留大量补丁文件，可以手动删除。

操作步骤：

  1. Win + R 输入 C:\Windows\SoftwareDistribution\Download 回车
  2. Ctrl + A 全选文件，按 Shift + Delete 删除

✅ 效果：释放 1GB-5GB 空间。

五、清理临时文件

Windows 运行过程中会产生大量临时文件，可以手动删除。

操作步骤：

  1. Win + R，输入 %temp%，回车 → 选中所有文件，按 Shift + Delete 永久删除
  2. Win + R，输入 temp，回车 → 删除所有文件
  3. Win + R，输入 prefetch，回车 → 删除所有文件

✅ 效果：释放 1GB-5GB 空间。

六、移动或调整虚拟内存

没有用过虚拟内存的就跳过这一步，如果你不知道什么是虚拟内存也跳过。
虚拟内存（Pagefile.sys）默认存放在C盘，通常会占用几GB空间。可将其移动到D盘或其他盘。

操作步骤：

  1. 右键 “此电脑”，选择 属性 → 高级系统设置
  2. 进入 高级 选项卡，点击 性能 → 设置
  3. 切换到 高级，点击 更改虚拟内存
  4. 取消 自动管理所有驱动器的分页文件大小
  5. 选择C盘 → 选择 无分页文件 → 点击 设置
  6. 选择D盘（或其他盘） → 选择 系统管理的大小 → 点击 设置
  7. 点击 确定，重启电脑生效

✅ 效果：看你虚拟内存放了多少，实际上你去虚拟机里面WSL什么的自己删除一样。

七、删除休眠文件（hiberfil.sys）

休眠文件默认存放在C盘，占用几GB空间。如果不使用休眠功能，可以关闭它。

关闭休眠功能：

  1. Win + R，输入 cmd，现在停下不要回车或者点确定
  2. 按 Ctrl + Shift + Enter 以管理员身份运行
  3. 输入以下命令并回车：

✅ 效果：释放 4GB-16GB 空间（取决于内存大小）。

八、修改软件默认存储路径

许多软件（如微信、QQ、迅雷、Chrome）的下载文件默认保存在C盘，可修改存储路径。

示例：修改微信存储路径

  1. 打开 微信 → 设置
  2. 选择 文件管理
  3. 修改 存储路径 到 D盘或其他位置
  4. 清理不必要的聊天记录

✅ 效果：减少C盘存储压力，释放 非常多的GB 空间。

九、开启”存储感知”自动清理

Windows 10/11 自带存储感知功能，可定期清理临时文件、回收站文件、下载文件夹等垃圾数据。

操作步骤：

  1. Win + I 打开 设置 → 系统 → 存储
  2. 打开 存储感知 开关
  3. 点击 配置存储感知或立即运行
  4. 设置自动清理的频率，如”每天”或”每周”

✅ 效果：定期自动清理垃圾文件，无需手动操作。

十、无损扩展C盘空间

数据无价，在进行下面操作之前最好先备份或者确定没有重要文件

如果C盘空间严重不足，可通过无损分区调整来增加C盘存储空间。

操作步骤（Windows 自带方法）：

  1. 右键 此电脑 → 管理 → 磁盘管理
  2. 右键 D盘（或其他磁盘） → 压缩卷
  3. 输入要压缩的空间（建议10GB+），点击 确定
  4. 右键 C盘 → 扩展卷，分配刚释放的空间

✅ 效果：无需重装系统，即可扩大C盘存储空间，想扩大多少都行。

BUT! HOWEVER! 但是！

这只是理想情况下。因为磁盘扩展需要连续的未分配空间，如果这么分出来的空闲空间不在C盘相邻的位置，就需要调整磁盘顺序。
这一步通常是通过将位于空闲空间之前的其他分区向后移动实现，让D盘分出来的空闲空间向前移动至紧邻C盘的位置。就像“冒泡”或者换位置一样，两两交换把空闲空间平移到C盘相邻的位置。

下面有2个好用的方法

1. 易我分区大师（EaseUS Partition Master）

易我分区大师

这个适合小白但是愿意花钱的用户，淘宝买一个7天体验或者30天体验会员20-60元，下载之后他有教程引导。但是要花钱我就不推荐了。

1. Disk Genius

Disk Genius

这个不花钱，选择下载免费版就行。注意不要下载到你需要扩容的盘！就是你要扩容C盘，就别下到C盘。要拆分D盘，也别放到D盘。

1. 下载好之后，解压安装。会让你扫一个二维码关注公众号，关注一下。

2. 使用之前，最好把别的软件关掉，微信什么的。打开，看到下面这个界面

3. 选择“拆分分区”，输入你想C盘扩容的大小。（当然要比你当前的空余大小小。你不能一点不留）

4. 拆分之后，会出现显示为新加卷的盘。你可以选择“删除分区”给他格式化为空闲区域。因为只有灰色的空闲区域能平移。
5. 接下来我们进行平移，选择D盘的“调整分区大小”（你要是从E盘什么的拆分出去的就从E盘开始），把空闲区域拖动到前面，点击保存。

6. 一直平移，直到空闲区域在C盘边上，可以点击扩容，然后你发现C盘变大了，保存。

7. 这样你就完成了扩容！
